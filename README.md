Fitur
1. Daftar Hewan & Tumbuhan Endemik — menampilkan data spesies endemik dalam bentuk grid, dikelompokkan berdasarkan kategori (Hewan/Tumbuhan).
2. Pencarian — mencari spesies endemik berdasarkan nama.
3. Detail Spesies — menampilkan informasi lengkap suatu spesies (nama, famili, genus, asal, sebaran, dan deskripsi).
4. Favorit — menyimpan dan mengelola daftar spesies favorit pengguna.
5. Mode Gelap (Dark Mode) — mendukung tampilan terang dan gelap yang dapat diaktifkan melalui halaman Pengaturan.
6. Filter Region — menyaring daftar spesies berdasarkan wilayah persebaran di Indonesia.

- Teknologi yang Digunakan
Bahasa: Java
Platform: Android (Android Studio)
UI Components: Material Components, CardView, Fragment, RecyclerView/GridView
Penyimpanan Lokal: SharedPreferences (preferensi pengguna), SQLite/Room (data spesies endemik)
Tema: Mendukung Light & Dark Mode melalui resource values dan values-night
